- node.js的安装(省略)
- cnpm 淘宝镜像的安装

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
- 以后npm换成cnpm
- 同步模块,通过sync命令马上同步一个模块,只有`cnpm`命令行才有这个功能

```
cnpm sync connect
```

- `npm install` 安装node_modules
